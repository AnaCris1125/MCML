Sitio web corporativo diseñado y desarrollado para una empresa
del sector construcción.

Caracteristicas del proyecto:

• Diseñar y desarrollar el sitio web corporativo de la empresa, alineado con su identidad visual y objetivos comerciales.
• Desarrollar una interfaz web moderna y visual, utilizando HTML, CSS y React, orientada a mejorar la experiencia de usuario y la usabilidad de la aplicación.
• Estructurar el proyecto para permitir la integración futura de módulos dinámicos.
• Realizar prototipos UI/UX en Figma y gestionar el código con GitHub.
